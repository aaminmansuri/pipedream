જ્યારે પૃષ્ઠ પ્રદર્શિત થશે ત્યારે બ્રાઉઝર વધારાની જગ્યાઓ અને વધારાની રેખાઓ દૂર કરશે.

 

ગમે તેટલી જગ્યાઓ અને કોઈપણ સંખ્યામાં નવી લાઈનો, માત્ર એક જ જગ્યા તરીકે ગણાય છે.

HTML ફોર્મેટિંગ તત્વો

અગાઉના પ્રકરણમાં, તમે HTML શૈલી વિશેષતાનો ઉપયોગ કરીને HTML સ્ટાઇલ વિશે શીખ્યા.

 

HTML એ ખાસ અર્થ સાથે ટેક્સ્ટને વ્યાખ્યાયિત કરવા માટે વિશેષ તત્વો પણ વ્યાખ્યાયિત કરે છે.

 

HTML એ બોલ્ડ અથવા ઇટાલિક ટેક્સ્ટ જેવા ફોર્મેટિંગ આઉટપુટ માટે <b> અને <i> જેવા તત્વોનો ઉપયોગ કરે છે.

 

ફોર્મેટિંગ તત્વોને વિશિષ્ટ પ્રકારના ટેક્સ્ટ પ્રદર્શિત કરવા માટે ડિઝાઇન કરવામાં આવ્યા હતા:

મહત્વપૂર્ણ ટેક્સ્ટ

ઇટાલિક ટેક્સ્ટ

લખાણ પર ભાર મૂક્યો

ચિહ્નિત ટેક્સ્ટ

નાનું લખાણ

કાઢી નાખેલ ટેક્સ્ટ

દાખલ કરેલ ટેક્સ્ટ

સબસ્ક્રિપ્ટ્સ

સુપરસ્ક્રિપ્ટ્સ

ઉદાહરણ

9.1 ટૂંકા અવતરણો માટે HTML <q>

HTML <q> ઘટક ટૂંકા અવતરણને વ્યાખ્યાયિત કરે છે.

 

બ્રાઉઝર્સ સામાન્ય રીતે <q> તત્વની આસપાસ અવતરણ ચિહ્નો દાખલ કરે છે.

<p>WWF's goal is to: <q>Build a future where people live in harmony with nature.</q></p>

 

9.2 લાંબા અવતરણો માટે HTML <blockquote>

HTML <blockquote> તત્વ અવતરણ કરેલ વિભાગને વ્યાખ્યાયિત કરે છે.

 

બ્રાઉઝર્સ સામાન્ય રીતે <blockquote> તત્વોને ઇન્ડેન્ટ કરે છે.

<p>Here is a quote from WWF's website:</p>
<blockquote cite="http://www.worldwildlife.org/who/index.html">
For 50 years, WWF has been protecting the future of nature.
The world's leading conservation organization,
WWF works in 100 countries and is supported by
1.2 million members in the United States and
close to 5 million globally.
</blockquote>

 

9.3 સંક્ષેપ માટે HTML <abbr>

HTML <abbr> તત્વ સંક્ષેપ અથવા ટૂંકાક્ષર વ્યાખ્યાયિત કરે છે.

 

સંક્ષિપ્ત શબ્દોને ચિહ્નિત કરવાથી બ્રાઉઝર, અનુવાદ પ્રણાલી અને શોધ-એન્જિનને ઉપયોગી માહિતી મળી શકે છે.

 

9.4 સંપર્ક માહિતી માટે HTML <address>

HTML <address> તત્વ દસ્તાવેજ અથવા લેખની સંપર્ક માહિતી (લેખક/માલિક) વ્યાખ્યાયિત કરે છે.

 

<address> ઘટક સામાન્ય રીતે ઇટાલિકમાં પ્રદર્શિત થાય છે. મોટાભાગના બ્રાઉઝર્સ એલિમેન્ટ પહેલા અને પછી લાઇન બ્રેક ઉમેરશે.

 

9.5 કાર્ય શીર્ષક માટે  HTML <cite>

HTML <cite> તત્વ કામના શીર્ષકને વ્યાખ્યાયિત કરે છે.

 

બ્રાઉઝર્સ સામાન્ય રીતે ઇટાલિકમાં <cite> તત્વો દર્શાવે છે.

 

9.6 HTML <bdo> બાય-ડાયરેક્શનલ ઓવરરાઇડ માટે

HTML <bdo> તત્વ દ્વિ-દિશાત્મક ઓવરરાઇડને વ્યાખ્યાયિત કરે છે.

 

<bdo> એલિમેન્ટનો ઉપયોગ વર્તમાન ટેક્સ્ટ દિશાને ઓવરરાઇડ કરવા માટે થાય છે

પ્રકરણ 11 HTML ટિપ્પણીઓ

 

HTML ટિપ્પણી ટૅગ્સ

તમે નીચેના વાક્યરચનાનો ઉપયોગ કરીને તમારા HTML સ્ત્રોતમાં ટિપ્પણીઓ ઉમેરી શકો છો:

ટિપ્પણીઓ બ્રાઉઝર દ્વારા પ્રદર્શિત થતી નથી, પરંતુ તે તમારા HTMLને દસ્તાવેજ કરવામાં મદદ કરી શકે છે.

ટિપ્પણીઓ વડે તમે તમારા HTML માં સૂચનાઓ અને રીમાઇન્ડર્સ મૂકી શકો છો

 

HTML ડીબગ કરવા માટે ટિપ્પણીઓ પણ શ્રેષ્ઠ છે, કારણ કે તમે ભૂલો શોધવા માટે, એક સમયે એક, કોડની HTML રેખાઓ પર ટિપ્પણી કરી શકો છો.

 

11.1 શરતી ટિપ્પણીઓ

તમે HTML માં શરતી ટિપ્પણીઓ પર ઠોકર ખાઈ શકો છો

 

સોફ્ટવેર પ્રોગ્રામ ટૅગ્સ

વિવિધ HTML સોફ્ટવેર પ્રોગ્રામ્સ દ્વારા HTML ટિપ્પણીઓ ટૅગ્સ પણ જનરેટ કરી શકાય છે.

 

ઉદાહરણ તરીકે <!--webbot bot--> ટૅગ્સ ફ્રન્ટપેજ અને એક્સપ્રેશન વેબ દ્વારા HTML ટિપ્પણીઓમાં આવરિત.

 

એક નિયમ તરીકે, આ ટૅગ્સને રહેવા દો, જે તેમને બનાવનાર સૉફ્ટવેરને સમર્થન આપે છે.

 

પ્રકરણ 12 HTML રંગો

 

12.1 રંગના નામ

CSS સાથે, રંગના નામોનો ઉપયોગ કરીને રંગો સેટ કરી શકાય છે

 

RGB (લાલ, લીલો, વાદળી)

HTML સાથે, આ ફોર્મ્યુલાનો ઉપયોગ કરીને RGB રંગ મૂલ્યો સ્પષ્ટ કરી શકાય છે: rgb(લાલ, લીલો, વાદળી)

 

દરેક પરિમાણ (લાલ, લીલો અને વાદળી) 0 અને 255 વચ્ચેના રંગની તીવ્રતાને વ્યાખ્યાયિત કરે છે.

 

ઉદાહરણ તરીકે, rgb(255,0,0) લાલ તરીકે પ્રદર્શિત થાય છે, કારણ કે લાલ તેના ઉચ્ચતમ મૂલ્ય (255) પર સેટ છે અને અન્ય 0 પર સેટ છે. નીચે RGB મૂલ્યોને મિશ્રિત કરીને પ્રયોગ કરો

 

12.2 હેક્સાડેસિમલ રંગો

HTML સાથે, RGB મૂલ્યો ફોર્મમાં હેક્સાડેસિમલ રંગ મૂલ્યોનો ઉપયોગ કરીને પણ સ્પષ્ટ કરી શકાય છે: #RRGGBB, જ્યાં RR (લાલ), GG (લીલો) અને BB (વાદળી) 00 અને FF (દશાંશ 0-255 સમાન) વચ્ચેના હેક્સાડેસિમલ મૂલ્યો છે. .

 

ઉદાહરણ તરીકે, #FF0000 લાલ તરીકે પ્રદર્શિત થાય છે, કારણ કે લાલ તેના ઉચ્ચતમ મૂલ્ય (FF) પર સેટ છે અને અન્ય સૌથી નીચા મૂલ્ય (00) પર સેટ છે.

 

બધા 3 પ્રકાશ સ્ત્રોતો માટે સમાન મૂલ્યોનો ઉપયોગ કરીને ગ્રેના શેડ્સને ઘણીવાર વ્યાખ્યાયિત કરવામાં આવે છે.

 

પ્રકરણ 13 HTML શૈલીઓ - CSS

 

13.1 CSS સાથે HTML સ્ટાઇલ

CSS એટલે કેસ્કેડીંગ સ્ટાઇલ શીટ્સ

 

HTML તત્વોમાં 3 રીતે સ્ટાઇલ ઉમેરી શકાય છે:

 

ઇનલાઇન - HTML ઘટકોમાં શૈલી વિશેષતાનો ઉપયોગ કરીને

આંતરિક - HTML <head> વિભાગમાં <style> ઘટકનો ઉપયોગ કરીને

બાહ્ય - એક અથવા વધુ બાહ્ય CSS ફાઇલોનો ઉપયોગ કરીને

સ્ટાઇલ ઉમેરવાની સૌથી સામાન્ય રીત, સ્ટાઇલને અલગ CSS ફાઇલોમાં રાખવાની છે. પરંતુ, આ ટ્યુટોરીયલમાં, અમે આંતરિક શૈલીનો ઉપયોગ કરીએ છીએ, કારણ કે તે દર્શાવવું સરળ છે, અને તમારા માટે તેને જાતે અજમાવવાનું સરળ છે.

 

13.2 ઇનલાઇન સ્ટાઇલ (ઇનલાઇન CSS)

ઇનલાઇન સ્ટાઇલનો ઉપયોગ એક HTML ઘટક પર અનન્ય શૈલી લાગુ કરવા માટે થાય છે:

 

ઇનલાઇન સ્ટાઇલ શૈલી વિશેષતાનો ઉપયોગ કરે છે.

 

આ ઉદાહરણ <h1> ઘટકના ટેક્સ્ટ રંગને વાદળીમાં બદલી નાખે છે.

 

આંતરિક શૈલી (આંતરિક CSS)

આંતરિક સ્ટાઇલનો ઉપયોગ એક HTML પૃષ્ઠ માટે શૈલીને વ્યાખ્યાયિત કરવા માટે થાય છે.

 

આંતરિક શૈલીને HTML પૃષ્ઠના <head> વિભાગમાં, <style> તત્વની અંદર વ્યાખ્યાયિત કરવામાં આવે છે:

 

ઉદાહરણ

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color:lightgrey;}
h1   {color:blue;}
p    {color:green;}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

 

13.3 બાહ્ય શૈલી (બાહ્ય CSS)

બાહ્ય શૈલી શીટનો ઉપયોગ ઘણા પૃષ્ઠો માટે શૈલીને વ્યાખ્યાયિત કરવા માટે થાય છે.

 

બાહ્ય સ્ટાઈલ શીટ વડે, તમે એક ફાઈલ બદલીને આખી વેબસાઈટનો દેખાવ બદલી શકો છો!

 

બાહ્ય શૈલી શીટનો ઉપયોગ કરવા માટે, HTML પૃષ્ઠના <head> વિભાગમાં તેની લિંક ઉમેરો:

 

ઉદાહરણ

<!DOCTYPE html>

<html>

<head>

  <link rel="stylesheet" href="styles.css">

</head>

<body>

 

<h1>This is a heading</h1>

<p>This is a paragraph.</p>

 

</body>

</html>

 

 

13.4 બાહ્ય શૈલી શીટ કોઈપણ ટેક્સ્ટ એડિટરમાં લખી શકાય છે. ફાઇલમાં કોઈપણ html ટૅગ્સ ન હોવા જોઈએ. સ્ટાઈલ શીટ ફાઈલ .css એક્સ્ટેંશન સાથે સેવ થવી જોઈએ.

 

"styles.css" કેવી દેખાય છે તે અહીં છે:

 

body {
    background-color: lightgrey;
}

h1 {
    color: blue;
}

p {
    color:green;
}

 

13.5CSS ફોન્ટ્સ

CSS કલર પ્રોપર્ટી HTML એલિમેન્ટ માટે ઉપયોગમાં લેવાતા ટેક્સ્ટ રંગને વ્યાખ્યાયિત કરે છે.

 

CSS ફોન્ટ-ફેમિલી પ્રોપર્ટી HTML એલિમેન્ટ માટે ઉપયોગમાં લેવાતા ફોન્ટને વ્યાખ્યાયિત કરે છે.

 

CSS ફોન્ટ-સાઇઝ પ્રોપર્ટી HTML એલિમેન્ટ માટે ઉપયોગમાં લેવાતા ટેક્સ્ટના કદને વ્યાખ્યાયિત કરે છે.

 

ઉદાહરણ

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
    color: blue;
    font-family: verdana;
    font-size: 300%;
}
p  {
    color: red;
    font-family: courier;
    font-size: 160%;
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

 

13.6 CSS બોક્સ મોડલ

દરેક HTML ઘટકની આસપાસ એક બૉક્સ હોય છે, પછી ભલે તમે તેને જોઈ ન શકો.

 

CSS બોર્ડર પ્રોપર્ટી HTML એલિમેન્ટની આસપાસ દૃશ્યમાન બોર્ડરને વ્યાખ્યાયિત કરે છે:

 

ઉદાહરણ

p {
    border: 1px solid black;
}

CSS પેડિંગ પ્રોપર્ટી સરહદની અંદર પેડિંગ (સ્પેસ) વ્યાખ્યાયિત કરે છે:

 

ઉદાહરણ

p {
    border: 1px solid black;
    padding: 10px;
}

 

CSS માર્જિન પ્રોપર્ટી સરહદની બહાર માર્જિન (જગ્યા) વ્યાખ્યાયિત કરે છે:

 

ઉદાહરણ

p {
    border: 1px solid black;
    padding: 10px;
    margin: 30px;
}

 

13.7 આઈડી એટ્રીબ્યુટ

ઉપરોક્ત તમામ ઉદાહરણો સામાન્ય રીતે HTML તત્વોને સ્ટાઇલ કરવા માટે CSS નો ઉપયોગ કરે છે.

 

એક વિશિષ્ટ તત્વ માટે વિશિષ્ટ શૈલીને વ્યાખ્યાયિત કરવા માટે, પ્રથમ તત્વમાં id વિશેષતા ઉમેરો:

 

<p id="p01">I am different</p>

પછી ચોક્કસ id સાથે તત્વ માટે શૈલી વ્યાખ્યાયિત કરો:

 

ઉદાહરણ

#p01 {

    Color:blue;

}

 

 

13.8 વર્ગ વિશેષતા

તત્વોના વિશિષ્ટ પ્રકાર (વર્ગ) માટે શૈલીને વ્યાખ્યાયિત કરવા માટે, તત્વમાં વર્ગ વિશેષતા ઉમેરો:

 

<p class="error">I am different</p>

હવે તમે વિશિષ્ટ વર્ગ સાથે તત્વો માટે એક અલગ શૈલી વ્યાખ્યાયિત કરી શકો છો:

 

ઉદાહરણ

p.error {
    color: red;
}

પ્રકરણ સારાંશ

ઇનલાઇન સ્ટાઇલ માટે HTML શૈલી વિશેષતાનો ઉપયોગ કરો

આંતરિક CSS ને વ્યાખ્યાયિત કરવા માટે HTML <style> ઘટકનો ઉપયોગ કરો

બાહ્ય CSS ફાઇલનો સંદર્ભ આપવા માટે HTML <link> ઘટકનો ઉપયોગ કરો

<style> અને <link> તત્વોને સંગ્રહિત કરવા માટે HTML <head> ઘટકનો ઉપયોગ કરો

ટેક્સ્ટ રંગો માટે CSS રંગ ગુણધર્મનો ઉપયોગ કરો

ટેક્સ્ટ ફોન્ટ્સ માટે CSS ફોન્ટ-ફેમિલી પ્રોપર્ટીનો ઉપયોગ કરો

ટેક્સ્ટના કદ માટે CSS ફોન્ટ-સાઇઝ પ્રોપર્ટીનો ઉપયોગ કરો

દૃશ્યમાન તત્વ સરહદો માટે CSS બોર્ડર પ્રોપર્ટીનો ઉપયોગ કરો

બોર્ડરની અંદર જગ્યા માટે CSS પેડિંગ પ્રોપર્ટીનો ઉપયોગ કરો

સરહદની બહારની જગ્યા માટે CSS માર્જિન પ્રોપર્ટીનો ઉપયોગ છે
